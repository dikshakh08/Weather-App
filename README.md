# Weather-App
Designed and developed a fully responsive weather application featuring real-time weather data, geolocation support, city search, dynamic weather visuals, and multi-day forecasting using JavaScript and OpenWeather API.
✨ Features
🔍 Search weather by city name
📍 Automatic current location detection
🌡️ Real-time temperature updates
💧 Humidity information
🌬️ Wind speed display
🌤️ Dynamic weather icons based on weather conditions
📅 5-Day Weather Forecast
📱 Fully Responsive Design
⚠️ Error handling for invalid city names
🛠️ Technologies Used
HTML5
CSS3
JavaScript (ES6)
OpenWeather API
